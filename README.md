connect
======

Command Line Tool for launching URI-scheme (e.g. mysql, mongo) clients
from URI-formatted addresses.

e.g.
takes: `postgres://tombrady:forlife@patriots.com:222/foxborough`
and opens the psql

# Support
* mongo
* mysql
* postgresql

# Usage:
```
npm install
chmod u+x connect
./connect
```
optional: add to PATH
